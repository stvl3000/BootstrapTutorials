Basic Table classes: .table, .table-striped, .table-bordered, .table-hover

.table class provides light padding and only horizontal dividers to a table.

.table-striped class adds zebra-stripes to the table.

.table-bordered class adds borders on all sides of cells.

.table-hover class enables a hover state on table rows.

.table-condensed class cut cell padding in Half.

contextual classes color the rows.the classes are: .active, .success, .info, .warning, .danger
