Collapsible Navbar

The Collapsible Navbar is hidden on small screens and replaced by a button in the top right corner when resizing this window. Only when the button is clicked, the navigation bar will be displayed in stacked mode.
However, the button is created by 3 off <span class="icon-bar"></span>, each of it created one line in the icon, which means it is a drawing rather than an Icon.
